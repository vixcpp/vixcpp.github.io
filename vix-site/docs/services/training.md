# Training

Structured training programs for teams and engineers who want to master Vix.cpp and modern C++ backend architecture.

---

## Core Training Tracks

### 1. Modern C++ Backend Foundations

- Memory model and performance fundamentals
- Coroutines and async design
- Event loops and schedulers
- HTTP and WebSocket internals
- Clean modular architecture

---

### 2. Async & Concurrency Mastery

- Coroutine lifecycle
- io_context deep dive
- Thread pools and CPU offloading
- Structured concurrency (when_all / when_any)
- Signal handling and graceful shutdown

---

### 3. Database & ORM Systems

- Connection pooling design
- Transactions and Unit of Work
- Repository patterns
- Migrations strategy
- Query performance optimization

---

### 4. P2P & Offline-First Architecture

- Node configuration
- Runtime lifecycle
- Peer discovery patterns
- Reliability and failure handling
- Observability strategies
